## 登录说明

1. 前端登录页地址：http://localhost:5173/login
2. 登录成功后系统会自动保存 JWT 到浏览器本地存储
3. 管理员登录后可进入管理后台和特色分类管理页面
4. 普通用户可进入发布商品、我的商品、我的订单等页面
